# Imaginable Things — Version 3.0

## Main upgrade: Premium Portfolio

Each project can now have:

- One main image
- Multiple additional images
- A full project description
- Services or techniques used
- Featured status
- Show/hide control
- Custom display order

When a visitor clicks a project, a professional full-screen project view opens with:

- Large image gallery
- Previous and next controls
- Thumbnail navigation
- Project information
- A button to request something similar

The quote button automatically places the project name into the quote form.

## Install

Before replacing files, run this inside your local project folder:

```bat
git pull origin main
```

Then replace the existing files with the contents of this version and run:

```bat
git add -A
git commit -m "Imaginable Things version 3.0 premium portfolio"
git push origin main
```

## Pages CMS

Open `Portfolio Projects`.

Each project now includes:

- Additional project images
- Services or techniques
- Show on website
- Display order

Lower display-order numbers appear first.
