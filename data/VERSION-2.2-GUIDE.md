# Imaginable Things — Version 2.2

## New features

- Structured quote request form
- The form prepares a complete WhatsApp message
- Editable FAQ section
- Editable Facebook, Instagram and TikTok links
- Editable project types in the quote form
- No paid service or database required

## Install

Copy all files into the local folder connected to GitHub and replace existing files.

Then run:

```bat
git add -A
git commit -m "Imaginable Things version 2.2"
git push
```

After Vercel says Ready, reopen Pages CMS.

A new section will appear:

`FAQs, Social Links & Quote Form`

Use it to enter the official social URLs, edit the FAQs and change the project options.
