# Imaginable Things — Version 6.2

## Content Manager

This version makes the remaining homepage content editable from Pages CMS on a phone.

### Editable sections
- Featured Projects heading and description
- Services heading
- Portfolio heading
- Testimonials heading
- How It Works heading and steps
- FAQ heading
- Quote section label
- Testimonials with optional photo/logo, star rating, visibility, and order
- FAQs with visibility and order

### Daily workflow
Open `/admin`, sign in to Pages CMS, edit the desired section, and save. Pages CMS commits the change to GitHub and Vercel publishes it automatically. No code editing is required for routine content changes.
