# Imaginable Things — Version 2.1

Esta versión amplía el panel visual.

## Ahora puedes editar desde Pages CMS

- Proyectos de la galería
- Texto principal del Hero
- Video y foto de portada
- Botones del Hero
- Historia del negocio
- Teléfono, WhatsApp, correo y ubicación
- Testimonios de clientes

## Instalar

Copia todos los archivos de esta carpeta dentro de la carpeta conectada a GitHub y reemplaza los archivos existentes.

Después ejecuta:

```bat
git add -A
git commit -m "Imaginable Things version 2.1"
git push
```

Cuando Vercel diga Ready, abre Pages CMS de nuevo. En el menú aparecerán:

- Portfolio Projects
- Homepage Content
- Testimonials

## Publicar un testimonio

1. Abre Testimonials.
2. Añade un item.
3. Escribe nombre, negocio y comentario.
4. Activa Show on website.
5. Pulsa Save.

## Cambiar el Hero

1. Abre Homepage Content.
2. Expande Hero Section.
3. Cambia el texto, video o poster.
4. Pulsa Save.
5. Espera que Vercel publique el cambio.
