# Imaginable Things — Version 5.1 Portfolio Pro

## What changed

- Featured Projects now comes directly from Portfolio Manager.
- Portfolio filters are created automatically from published categories.
- Every project supports a cover photo plus multiple gallery photos.
- Project detail includes item type, technique, production time and completion date.
- Optional project video support.
- Photo count appears on portfolio cards.
- Related projects remain available inside the project viewer.
- “Request Something Similar” continues to prefill the quote details.
- Duplicate/broken project modal markup was removed.

## Publish a project from your phone

1. Visit `/admin` and open Pages CMS.
2. Open **Portfolio Manager**.
3. Add or edit a project.
4. Choose the cover photo and add gallery photos.
5. Complete the optional Portfolio Pro fields.
6. Set **Published on website** to true.
7. Set **Feature on website** to true only for your strongest work.
8. Press **Save** and wait for Vercel to deploy.

## Recommended photo order

1. Full finished product.
2. Close-up of the decoration.
3. Alternate angle or back view.
4. Process photo.
5. Packaging or final presentation.
