# Version 6.3 — Imaginable OS Dashboard

## What changed

- Rebuilt `/admin` as the Imaginable OS dashboard.
- Added live counts for visible projects, featured projects, active services and published testimonials.
- Added mobile-friendly quick actions for the main Pages CMS sections.
- Added automatic business setup checks for WhatsApp, email, logo and FAQs.
- Added system/version information and a direct link to the public website.

## Files in this update

- `admin/index.html`
- `admin/dashboard.js`
- `ROADMAP.md`
- `VERSION-6.3-GUIDE.md`

## Test after deployment

1. Open `/admin` in a private/incognito browser window.
2. Confirm that the four content totals appear.
3. Confirm that the setup checks show Ready or Review.
4. Tap **Open Content Manager** and sign in to Pages CMS.
5. Test the layout from a phone.

The dashboard is a secure landing page, while content editing continues inside Pages CMS.
