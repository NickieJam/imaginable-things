# Imaginable Things — Phase 1 Admin Guide

## Admin address

Open:

`https://www.imaginablethingsllc.com/admin`

The page opens the hosted Pages CMS editor. Pages CMS edits the website content directly in GitHub, and Vercel publishes the saved changes.

## First login

1. Tap **Open Admin Panel**.
2. Sign in with the GitHub account that owns `NickieJam/imaginable-things`.
3. Open the `imaginable-things` repository.
4. Select the `main` branch.
5. Choose **Portfolio Manager**.

## Add a portfolio project from iPhone

1. Open **Portfolio Manager**.
2. Add a new item.
3. Enter the project name and category.
4. Upload one **Cover photo**.
5. Add more pictures under **Gallery photos**.
6. Add the description and services used.
7. Leave **Published on website** turned on.
8. Save.

Vercel normally updates the public site after the new GitHub commit is deployed.

## Add the admin to the iPhone Home Screen

1. Open `/admin` in Safari.
2. Tap the Share icon.
3. Tap **Add to Home Screen**.
4. Name it `Imaginable Admin`.
