# Imaginable Things — Version 5.2

## Smart Quote Wizard 2.0

This update improves the quote workflow without changing the CMS or portfolio data.

### New features
- Product-aware placement choices.
- Multiple design placements in one request.
- Exact quantity field in addition to quantity ranges.
- Question about who supplies the garments/items.
- Preferred contact method.
- Mobile-friendly camera/photo upload wording.
- Full review before opening WhatsApp.
- Better WhatsApp message with all project details.
- Removed an unused duplicate quote section from the homepage HTML.

### Test checklist
1. Open Get a Quote.
2. Complete all nine steps.
3. Select more than one placement.
4. Upload and remove an image or embroidery file.
5. Confirm the review screen.
6. Open WhatsApp and confirm the message contains every answer.
